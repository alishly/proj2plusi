## Построение бинарного дерева по двоичным путям
Цель: Разработка программы для построения бинарного дерева целых чисел по заданным в файле двоичным путям.
Задачи (Функциональные требования):
- Считывание данных из текстового файла
- Построение бинарного дерева
- Проверка корректности входных данных
- Поддержка отрицательных чисел и обработка переполнения int.
- Вывод дерева на экран в повёрнутом на 90° виде.
- Устойчивость к ошибкам
